# چک‌لیست بررسی دقیق (Due Diligence Checklist)

**نسخه:** ۱.۰ | **تاریخ:** بهمن ۱۴۰۴ | **برای:** [نام سرمایه‌گذار]

---

## مقدمه

این سند، چک‌لیستی از اطلاعات و اسناد مورد نیاز برای انجام فرآیند بررسی دقیق (Due Diligence) بر روی **شرکت فناوری سلامت آروم‌شو** ("شرکت") را ارائه می‌دهد. هدف از این فرآیند، راستی‌آزمایی اطلاعات ارائه شده توسط شرکت، شناسایی ریسک‌های بالقوه و تایید فرضیات سرمایه‌گذاری قبل از نهایی شدن قرارداد است. از شرکت درخواست می‌شود تا اسناد و اطلاعات زیر را در یک اتاق داده مجازی (Virtual Data Room) سازماندهی و در دسترس تیم سرمایه‌گذار قرار دهد.

## بخش ۱: اطلاعات شرکتی و حقوقی

| ردیف | موضوع | اسناد مورد نیاز |
| :--- | :--- | :--- |
| ۱.۱ | **اساسنامه و مدارک ثبت** | - آخرین نسخه اساسنامه شرکت<br>- روزنامه رسمی آگهی تاسیس و آخرین تغییرات<br>- گواهی ثبت شرکت و شناسه ملی |
| ۱.۲ | **ساختار سهامداری** | - جدول سهامداران (Cap Table) دقیق و به‌روز<br>- کلیه قراردادهای سهامداری، واگذاری سهام (Vesting) و اختیار خرید (Option) موجود |
| ۱.۳ | **صورتجلسات** | - صورتجلسات هیئت مدیره و مجامع عمومی از زمان تاسیس |
| ۱.۴ | **مالکیت معنوی (IP)** | - لیست تمام دارایی‌های فکری (نرم‌افزار، برند، دامنه)<br>- مدارک ثبت برند تجاری (در صورت وجود)<br>- قراردادهای واگذاری مالکیت معنوی از کارکنان و مشاوران به شرکت |
| ۱.۵ | **قراردادهای کلیدی** | - قراردادهای استخدام مدیران کلیدی<br>- قراردادهای همکاری با تامین‌کنندگان و شرکای اصلی<br>- قرارداد اجاره دفتر |
| ۱.۶ | **دعاوی حقوقی** | - لیستی از هرگونه دعوی حقوقی، تحقیق یا بازرسی در جریان یا مختومه علیه شرکت |
| ۱.۷ | **بیمه** | - بیمه‌نامه‌های مسئولیت، دارایی‌ها و کارکنان شرکت |

## بخش ۲: اطلاعات مالی

| ردیف | موضوع | اسناد مورد نیاز |
| :--- | :--- | :--- |
| ۲.۱ | **صورت‌های مالی** | - صورت‌های مالی حسابرسی شده یا مدیریتی برای تمام دوره‌های فعالیت<br>- ترازنامه، صورت سود و زیان، صورت جریان نقدینگی |
| ۲.۲ | **مدل مالی** | - فایل اکسل مدل مالی با تمام فرمول‌ها و مفروضات |
| ۲.۳ | **حساب‌های بانکی** | - صورتحساب‌های بانکی ۶ ماه گذشته |
| ۲.۴ | **مالیات** | - اظهارنامه‌های مالیاتی و گواهی پرداخت یا ترتیب پرداخت بدهی‌های مالیاتی |
| ۲.۵ | **بدهی‌ها** | - لیست تمام بدهی‌ها و وام‌های شرکت به همراه قراردادهای مربوطه |

## بخش ۳: اطلاعات محصول و فناوری

| ردیف | موضوع | اسناد مورد نیاز |
| :--- | :--- | :--- |
| ۳.۱ | **معماری فنی** | - سند معماری فنی دقیق<br>- دیاگرام‌های زیرساخت و جریان داده |
| ۳.۲ | **کد منبع (Source Code)** | - دسترسی به مخزن کد (GitHub, GitLab) برای بررسی کیفیت کد و فرآیندهای توسعه |
| ۳.۳ | **امنیت** | - گزارش تست نفوذ (Penetration Test) (در صورت وجود)<br>- سیاست‌ها و رویه‌های امنیتی شرکت |
| ۳.۴ | **نقشه راه محصول** | - سند نقشه راه محصول با جزئیات کامل |
| ۳.۵ | **تحلیل و داده** | - دسترسی به ابزارهای تحلیلی (مانند Google Analytics, Mixpanel) برای بررسی معیارهای محصول و تعامل کاربران |

## بخش ۴: اطلاعات تیم و پرسنل

| ردیف | موضوع | اسناد مورد نیاز |
| :--- | :--- | :--- |
| ۴.۱ | **ساختار سازمانی** | - چارت سازمانی به‌روز<br>- لیست تمام کارکنان به همراه سمت، تاریخ شروع به کار و حقوق |
| ۴.۲ | **قراردادهای پرسنلی** | - نمونه قرارداد استخدام استاندارد شرکت<br>- قراردادهای عدم رقابت و محرمانگی امضا شده توسط کارکنان |
| ۴.۳ | **برنامه سهام تشویقی (ESOP)** | - سند برنامه ESOP<br>- لیست تمام گزینه‌های سهام واگذار شده و باقی‌مانده |
| ۴.۴ | **سوابق بنیان‌گذاران** | - رزومه کامل بنیان‌گذاران و مدیران کلیدی<br>- انجام بررسی سوابق (Background Check) |

## بخش ۵: اطلاعات بازار و فروش

| ردیف | موضوع | اسناد مورد نیاز |
| :--- | :--- | :--- |
| ۵.۱ | **تحلیل بازار** | - گزارش‌های تحلیل بازار، رقبا و اندازه بازار |
| ۵.۲ | **استراتژی بازاریابی** | - سند برنامه ورود به بازار (Go-To-Market Plan)<br>- گزارش عملکرد کمپین‌های بازاریابی گذشته |
| ۵.۳ | **قیف فروش** | - داده‌های مربوط به قیف فروش (از آگاهی تا تبدیل) و نرخ‌های تبدیل هر مرحله |
| ۵.۴ | **مشتریان** | - لیست ۱۰ مشتری یا کاربر برتر (به صورت ناشناس در صورت لزوم)<br>- نتایج نظرسنجی‌ها و مصاحبه‌های انجام شده با مشتریان |
| ۵.۵ | **قیمت‌گذاری** | - تاریخچه و تحلیل استراتژی قیمت‌گذاری |
