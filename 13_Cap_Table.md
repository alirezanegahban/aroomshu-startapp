# جدول سهامداران (Capitalization Table)

**نسخه:** ۱.۰ | **تاریخ:** بهمن ۱۴۰۴ | **محرمانه**

---

## ۱. مقدمه

این سند، جدول سهامداران (Cap Table) استارتاپ "آروم‌شو" را در مراحل مختلف، قبل و بعد از راند سرمایه‌گذاری Seed، نمایش می‌دهد. Cap Table یک سند حیاتی است که ساختار مالکیت شرکت را به تفصیل نشان می‌دهد و برای مدیریت سهام، برنامه‌ریزی برای راندهای آتی سرمایه‌گذاری و تصمیم‌گیری‌های استراتژیک ضروری است.

## ۲. مفروضات راند سرمایه‌گذاری Seed

-   **ارزش‌گذاری پیش از سرمایه‌گذاری (Pre-Money Valuation):** ۱۵ میلیارد تومان
-   **مبلغ سرمایه‌گذاری:** ۵ میلیارد تومان
-   **ارزش‌گذاری پس از سرمایه‌گذاری (Post-Money Valuation):** ۲۰ میلیارد تومان (۱۵ میلیارد + ۵ میلیارد)
-   **سهام واگذار شده به سرمایه‌گذاران جدید:** ۵ میلیارد / ۲۰ میلیارد = **۲۵٪**
-   **ایجاد سهام تشویقی جدید (New ESOP Pool):** ۱۰٪ از سهام پس از راند سرمایه‌گذاری (Post-Money)

## ۳. جدول سهامداران قبل از راند Seed (Pre-Seed Stage)

در این مرحله، شرکت تنها توسط بنیان‌گذاران و یک استخر اولیه سهام تشویقی (ESOP) کنترل می‌شود.

| سهامدار | نوع سهام | تعداد سهام | درصد مالکیت |
| :--- | :--- | :--- | :--- |
| دکتر سارا احمدی (CEO) | عادی (Common) | ۴,۵۰۰,۰۰۰ | ۴۵.۰٪ |
| امیر رضایی (CTO) | عادی (Common) | ۴,۵۰۰,۰۰۰ | ۴۵.۰٪ |
| استخر سهام تشویقی (ESOP) | اختیار خرید (Option) | ۱,۰۰۰,۰۰۰ | ۱۰.۰٪ |
| **مجموع** | | **۱۰,۰۰۰,۰۰۰** | **۱۰۰.۰٪** |

## ۴. جدول سهامداران پس از راند Seed (Post-Seed Stage)

این جدول، ساختار مالکیت را پس از ورود سرمایه‌گذاران جدید و افزایش استخر سهام تشویقی نشان می‌دهد.

### ۴.۱. محاسبه تعداد سهام جدید

برای حفظ سادگی، فرض می‌کنیم قیمت هر سهم در راند Seed بر اساس ارزش‌گذاری Pre-Money تعیین می‌شود:

-   **قیمت هر سهم:** ۱۵,۰۰۰,۰۰۰,۰۰۰ تومان / ۱۰,۰۰۰,۰۰۰ سهم = **۱,۵۰۰ تومان**
-   **تعداد سهام جدید برای سرمایه‌گذاران:** ۵,۰۰۰,۰۰۰,۰۰۰ تومان / ۱,۵۰۰ تومان = **۳,۳۳۳,۳۳۳ سهم**

حال باید استخر جدید ESOP را محاسبه کنیم. هدف ما ۱۰٪ از کل سهام Post-Money است. بگذارید `X` تعداد کل سهام جدید (برای سرمایه‌گذار و ESOP) باشد.

-   `کل سهام Post-Money = سهام فعلی + X`
-   `سهام جدید ESOP = 0.10 * (10,000,000 + X)`
-   `X = سهام سرمایه‌گذار + سهام جدید ESOP`
-   `X = 3,333,333 + 0.10 * (10,000,000 + X)`
-   `X = 3,333,333 + 1,000,000 + 0.10X`
-   `0.90X = 4,333,333`
-   `X = 4,814,815`

-   **سهام جدید ESOP:** `4,814,815 - 3,333,333 = 1,481,482`
-   **کل سهام Post-Money:** `10,000,000 + 4,814,815 = 14,814,815`

## ۵. تحلیل رقیق شدن سهام (Dilution Analysis)

این جدول نشان می‌دهد که سهام بنیان‌گذاران در این راند چگونه رقیق (Dilute) شده است:

| سهامدار | مالکیت Pre-Seed | مالکیت Post-Seed | میزان رقیق شدن |
| :--- | :--- | :--- | :--- |
| دکتر سارا احمدی | ۴۵.۰٪ | ۳۰.۴٪ | -۱۴.۶٪ |
| امیر رضایی | ۴۵.۰٪ | ۳۰.۴٪ | -۱۴.۶٪ |

این میزان رقیق شدن، یک امر طبیعی و ضروری برای جذب سرمایه و رشد شرکت است. با افزایش ارزش شرکت در آینده، ارزش دلاری سهام باقی‌مانده بنیان‌گذاران به مراتب بیشتر از ارزش اولیه سهام آن‌ها خواهد بود.

## ۶. سناریوی جایگزین (سرمایه‌گذار ۲۵٪ کامل را می‌خواهد)

در این سناریو، ارزش‌گذاری Post-Money شامل استخر جدید ESOP نیز می‌شود.

-   سهام ESOP جدید: ۱۰٪ = ۲ میلیارد تومان
-   ارزش سهام بنیان‌گذاران: ۶۵٪ = ۱۳ میلیارد تومان

| سهامدار | درصد مالکیت | ارزش سهام |
| :--- | :--- | :--- |

| سرمایه‌گذار Seed | ۲۵.۰٪ | ۵ میلیارد تومان |

این سناریو برای سرمایه‌گذار مطلوب‌تر است اما منجر به رقیق شدن بیشتر سهام بنیان‌گذاران می‌شود. انتخاب بین این دو سناریو بخشی از فرآیند مذاکره با سرمایه‌گذار است.
