## راهنمای استفاده

### برای سرمایه‌گذاران

اگر زمان محدودی دارید، توصیه می‌کنیم ابتدا این اسناد را مطالعه کنید:

1.  **Executive Summary (سند ۰۱)** - برای درک کلی کسب‌وکار
2.  **Pitch Deck (سند ۰۲)** - برای مرور سریع فرصت
3.  **Investment Memo (سند ۰۳)** - برای تحلیل عمیق‌تر
4.  **Financial Model (سند ۱۲)** - برای بررسی اقتصاد کسب‌وکار
5.  **Risk Analysis (سند ۱۷)** - برای درک چالش‌ها

### برای ارزیابی دانشگاهی

برای ارزیابی جامع یک پروژه دانشجویی، توصیه می‌شود تمام اسناد را به ترتیب شماره مطالعه کنید. این بسته تمام جنبه‌های یک طرح کسب‌وکار حرفه‌ای را پوشش می‌دهد.

### برای تیم داخلی

اسناد زیر به عنوان مرجع عملیاتی استفاده می‌شوند:

-   **Product Roadmap (سند ۰۹)** - برای تیم محصول
-   **Technical Architecture (سند ۱۱)** - برای تیم فنی
-   **Go-To-Market Plan (سند ۰۷)** - برای تیم بازاریابی و فروش
-   **Metrics & KPIs (سند ۱۸)** - برای تمام تیم‌ها
